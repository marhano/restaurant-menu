# Changelog

## [0.0.1] - 2026-04-24
### Added
- Initial release
- Menu browsing with categories and sub-categories
- Menu item cards with image, description, price
- Multi-section basket (configurable: Kitchen, Bar, ...)
- Auto-routing of items to basket sections via category/sub-category mapping
- Ellipsis menu on item card to override basket section
- Quantity controls (add same item to stack, +/- inside basket, remove)
- Table info header
- Edit basket item to add a note
- Next serving button and send order
- destroy() lifecycle method
